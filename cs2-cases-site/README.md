# CS2 Case Opening - Сайт открытия кейсов Counter-Strike 2

Современный веб-сайт для симуляции открытия кейсов CS2 с красивым дизайном и реалистичной анимацией.

## 🎮 Особенности

- **Реалистичная система вероятностей** - шансы выпадения предметов максимально приближены к реальной игре
- **Красивая анимация рулетки** - плавная анимация открытия кейсов с эффектом предвкушения
- **Система инвентаря** - все выигранные предметы сохраняются в инвентаре
- **Торговая система** - продавайте предметы за 80% от рыночной стоимости
- **Система апгрейда** - обменивайте предметы на более редкие с доплатой
- **Профиль игрока** - отслеживайте свои достижения и прогресс
- **Подробная статистика** - анализ всей игровой активности
- **Система достижений** - разблокируйте награды за различные действия
- **Адаптивный дизайн** - отлично работает на всех устройствах
- **Темная тема CS2** - современный дизайн в стиле Counter-Strike 2

## 🚀 Быстрый запуск

### Требования
- PHP 7.4 или выше
- Веб-сервер (Apache, Nginx или встроенный сервер PHP)

### Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd cs2-cases-site
   ```

2. **Запустите встроенный сервер PHP:**
   ```bash
   php -S localhost:8000
   ```

3. **Откройте в браузере:**
   ```
   http://localhost:8000
   ```

### Альтернативные способы запуска

#### Использование Apache/Nginx
1. Скопируйте файлы в папку веб-сервера (например, `/var/www/html/`)
2. Убедитесь, что PHP включен
3. Откройте сайт в браузере

#### Использование XAMPP/WAMP/MAMP
1. Поместите папку проекта в `htdocs` (XAMPP) или `www` (WAMP/MAMP)
2. Запустите Apache
3. Перейдите на `http://localhost/cs2-cases-site/`

## 🎯 Функциональность

### 📦 Система кейсов
- **Chroma Case** - 250 монет
- **Phoenix Case** - 200 монет  
- **Gamma Case** - 300 монет
- **Revolution Case** - 350 монет
- Реалистичная анимация рулетки с предпросмотром предметов

### 🎲 Система редкости
- **Consumer Grade** (79.92%) - обычные предметы (12-50 монет)
- **Industrial Grade** (15.98%) - промышленные предметы (80-120 монет)
- **Mil-Spec Grade** (3.2%) - военные предметы (350-580 монет)
- **Restricted** (0.64%) - ограниченные предметы (1,200-1,800 монет)
- **Classified** (0.32%) - секретные предметы (4,200-5,800 монет)
- **Covert** (0.64%) - скрытые предметы (15,000-25,000 монет)
- **Special Items** (0.26%) - ножи (28,000-45,000 монет)

### 🔫 Типы оружия
- **Пистолеты** (Glock, P250, Desert Eagle и др.)
- **Автоматы** (AK-47, M4A4, M4A1-S и др.)
- **Снайперские винтовки** (AWP, SSG 08 и др.)
- **Пистолеты-пулеметы** (MP9, P90, UMP-45 и др.)
- **Дробовики** (XM1014, Nova и др.)
- **Ножи** (Karambit, Bayonet, Butterfly и др.)

### 💰 Торговая система
- **Продажа предметов** за 80% от рыночной стоимости
- **Система апгрейда** - обмен предметов на более редкие
- **Комиссия 10%** за апгрейд
- **Ограничения апгрейда** - только на 1-2 уровня редкости выше

### 📊 Система статистики
- **Общая статистика** - баланс, инвентарь, прибыль/убыток
- **Торговая статистика** - продажи, апгрейды, заработок
- **Распределение по редкости** - графики и прогресс-бары
- **История операций** - детальная история продаж и апгрейдов

### 🏆 Система достижений
- **Первый предмет** - получите первый предмет
- **Коллекционер** - соберите 10 предметов  
- **Счастливчик** - получите нож
- **Торговец** - заработайте 1000 монет
- **Апгрейдер** - выполните 5 апгрейдов
- **Магнат** - инвентарь стоимостью 10,000 монет

## 🎨 Кастомизация

### Добавление новых предметов
Отредактируйте файл `config.php`:
```php
$items['restricted'][] = ['name' => 'Новое оружие | Скин', 'type' => 'Rifle'];
```

### Изменение вероятностей
Измените значения в массиве `$rarities` в `config.php`:
```php
$rarities['covert']['chance'] = 1.0; // Увеличить шанс Covert предметов
```

### Настройка баланса
Начальный баланс можно изменить в файлах `index.php` и `get_balance.php`:
```php
$_SESSION['balance'] = 50000; // Начальный баланс 50000 монет
```

## 🔧 Устранение неполадок

### Ошибки PHP
- Убедитесь, что включены сессии PHP
- Проверьте права доступа к файлам
- Включите отображение ошибок для отладки

### Проблемы с JavaScript
- Откройте консоль разработчика (F12)
- Проверьте наличие ошибок в Network вкладке
- Убедитесь, что AJAX запросы выполняются корректно

### Проблемы со стилями
- Очистите кеш браузера (Ctrl+F5)
- Проверьте загрузку CSS файлов
- Убедитесь в корректности путей к файлам
